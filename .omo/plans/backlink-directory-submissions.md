## Tips

- **Don't submit all at once** — spread across 4-6 weeks (5-10 per week)
- **Track in a spreadsheet** — note which ones approved, which rejected
- **Customize descriptions** — make each submission slightly unique
- **No keyword stuffing** — keep descriptions natural
- **Check approval** — some directories take days to approve listings
- **Revisit in 3 months** — some directories auto-expire listings
